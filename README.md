# my-forms

### Run local development

#### Prerequisites

- Node 23+
- pnpm (`npm i -g pnpm`)

```bash
# Install all dependencies
pnpm install

# Generate graphql types
pnpm run generate

# Start the development environment
pnpm run dev

# Access http://localhost:5173/
```

---

### Run with Docker

#### Prerequisites

- Docker and Docker compose

```bash
docker compose up

# Access http://localhost/
```
